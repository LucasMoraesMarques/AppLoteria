FUNCIONALIDADES

-> CADASTRO DE JOGOS

    Permite os seguintes tipos de entradas de jogos externos:

    1 -> Entrada manual por digitação
        Permite:
        -> Especificar o número de jogos
        -> O número de marcações por apostas

    2 -> Entrada por arquivo
        Permite:
        -> Arquivos do tipo csv, txt ou xlsx
        -> Especificar o número de jogos
        -> O número de marcações por apostas

    3 -> Entrada pela Área de transferência (ctrl + c)
        Permite:
        -> Leitura de jogos copiados seguindo os seguintes requisitos:
            -> Números separados por vírgula(sem espaço entre número e vírgula)
            -> Números separados por espaços
            -> Um jogo por linha
            -> Qualquer tamanho de jogo

-> GERADORES DE JOGOS

    1 -> GERADOR SIMPLES:
        -> Gera jogos randômicos de forma autônoma.
        -> Permite jogos com mais números por aposta
        -> Permite remoção e fixação de números

    2 -> GERADOR INTELIGENTE (Disponível apenas com download de arquivos externos)
        -> Gera jogos randômicos de forma personalizada, por meio de filtros específicos
        -> Permite filtros como:
            -> Máxima diferença entre dois números
            -> Máxima sequência ininterrupta
            -> Mínima sequência ininterrupta
            -> Paridade do jogo, isto é, se é par ou ímpar
            -> Quantidade de números primos
            -> Remoção e fixação de números
            -> Foi sorteado
        -> A única desvantagem é que só permite o número fixo padrão para cada aposta.
        -> Não está disponível para a Mega Sena

-> MANIPULAÇÃO DE JOGOS

    Permite as seguintes operações com conjuntos de jogos:

    1 -> Conferir resultados e exportar por arquivo
    2 -> Filtrar jogos
    3 -> Mostrar estatísticas
    4 -> Deletar jogos

-> DATABASE DE RESULTADOS

    Fornece as seguintes funcionalidades:

    1 -> Mostrar todos os resultados de uma determinada Loteria
    2 -> Mostrar estatísticas dos resultados
    3 -> Filtrar database de resultados
    4 -> Mostrar combinações que mais saíram
